关于

一、密语 web 服务有什么用？

密语 web 服务提供文本加解密服务，加密可以使您的信息无法被阅读，您就可以安全地在网络中传输隐私、财产信息，除您以外任何人都不能理解您所发送密文，除非他们知道您加密所用的密码和方法。

二、密语 web 服务 和 密语输入法有何关系？

密语 web 服务所提供的文本加解密服务，与密语输入法中的是一致的。目前，密语输入法是一款付费软件，且仅提供 iOS 版本。密语 web 服务可以免费使用，适用于任何系统。但是密语 web 服务的安全性、便捷性远不能与密语输入法相提并论，暂时也不提供文件加解密等诸多功能。我们始终建议您下载使用密语输入法。

三、密语 web 服务是安全的吗？

是的，本网站仅在您的浏览器中运行，不会将任何数据发送到网络中，也不会以任何方式存储您的数据，我们甚至不知道您访问过本站。

这是因为密语 web 是用纯 HTML + Javascript 编写，其运行不需要借助服务器。此外，本站还默认使用了https，以增强安全性。

但是，密语 web 服务并不能确保您的安全，您有“访问到冒牌密语 web 服务”的危险，即使您输入的网址是正确的，您使用的浏览器或您的设备中仍有可能存在木马病毒代码。如果您没有IT技术背景，我们始终建议您下载使用密语输入法。

四、加解密算法的说明

  1. 我们无法为您提供随机打乱的解密服务，因为是它是随机的，人类语言是如此的博大精深，以致于一句话中，任何一个字的改变，都可以带来完全不同的意思。
  2. 凯撒密码仅支持26个英文字母，因为这是正宗的凯撒密码。
  3. 摩尔斯电码不支持中文，因为这是正宗的摩尔斯电码。
  4. Emoji 表情加密暂不支持中文，因为本开发者还不具有这个能力：)
  5. AES加密与密语输入法 1.x 版本不兼容，请等待输入法更新。
